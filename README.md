# NIST CSF 2.0 & RMF Mapping Repository
This repository provides a structured and transparent mapping between the NIST Cybersecurity Framework (CSF) 2.0 and the NIST Risk Management Framework (RMF). The goal is to help organizations, security professionals, and researchers better understand how CSF 2.0 outcomes, categories, and functions align with RMF tasks, steps, and security controls.

By offering a consolidated and easy‑to‑navigate reference, this project supports cybersecurity program development, compliance initiatives, and governance activities that rely on both frameworks.

📘 Purpose
The main objectives of this repository are:

To provide a clear correlation between CSF 2.0 and RMF elements.

To support organizations in integrating CSF 2.0 into RMF‑based security programs.

To help professionals quickly identify which RMF tasks and controls support specific CSF outcomes.

To offer a public, community‑friendly resource that can evolve through contributions.

📂 Repository Contents
This repository includes:

Mapping spreadsheets correlating CSF 2.0 Functions, Categories, and Outcomes with RMF Steps and Tasks.

Reference tables connecting CSF 2.0 to NIST SP 800‑53 Rev. 5 control families.

Documentation explaining the methodology used to build the mapping.

Examples and guidance for applying the mapping in real‑world cybersecurity programs.

🔍 What This Mapping Helps You Do
Understand how CSF 2.0 can be operationalized using RMF processes.

Identify which RMF activities support CSF 2.0 strategic outcomes.

Improve alignment between governance, risk, and technical security controls.

Facilitate audits, assessments, and maturity evaluations.

Support the creation of Profiles, Implementation Plans, and Risk Strategies based on CSF 2.0.

🛠 Methodology
The mapping was developed by:

Analyzing official NIST publications, including CSF 2.0, SP 800‑37 Rev. 2, SP 800‑53 Rev. 5, and related guidance.

Identifying conceptual and operational relationships between CSF outcomes and RMF tasks.

Ensuring traceability between strategic objectives (CSF) and actionable security controls (RMF/SP 800‑53).

Structuring the mapping to be easy to read, modular, and expandable.
